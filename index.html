/create_user

Creates a user given the following parameters:

Example:

{

"username" : "brenwick"

"email" : "email@hi.com"

"phone" : "123-4567"

"password" : "notapassword"

"retypePassword" : "notapassword"

}



/login

"Logs in User" if password sent matches one in Database

Example:

{

"id" : "5f8f7d2d1dba2312101e01b5"

"password" : "01234"

}
